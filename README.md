# Fullstack ToDo app

## Installation

1. Klona projektet från GitHub till din lokala dator:

    ```bash
    git clone https://github.com/dens0n/fullstack-todo-app.git
    ```

2. Gå in i klientmappen:

    ```bash
    cd client
    ```

3. Installera alla nödvändiga beroenden:

    ```bash
    npm install
    ```

4. Starta utvecklingsservern:

    ```bash
    npm run dev
    ```

5. öppna en ny terminal

6. Gå in i servermappen:

    ```bash
    cd server
    ```

7. Installera alla nödvändiga beroenden:

    ```bash
    npm install
    ```

8. Starta utvecklingsservern:

    ```bash
    npm run dev
    ```
